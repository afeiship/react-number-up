# react-number-up
> Number up effect for react.

## installation
```shell
npm install -S @jswork/react-number-up
```

## usage
1. import css
  ```scss
  @import "~@jswork/react-number-up/dist/style.css";

  // or use sass
  @import "~@jswork/react-number-up/dist/style.scss";
  ```
2. import js
  ```js
__GENERATE_DAPP__
  ```
